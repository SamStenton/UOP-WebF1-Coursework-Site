Introduction

As a student of Computer Science I get access to a virtual machine. To access the machine I first need to switch it on via the web console. Once the VM status is showing "RUNNING" I can login to it using a dedicated SSH program or through my terminal by running the command ssh up778465@put.ip.address.here

Before being able to upload code to the VM and start running it I should be able to edit it first. Most operating systems come with a default text editor to edit code in but these can be very limited in functionality. Rich and Jack recommended that we used Atom which is an open source editor created by GitHub. For the past couple years my text editor of choice has been Sublime Text 3 which is a paid editor similar to Atom. I have used both editors extensively but have found myself getting frustrated at the sluggish nature of Atom compared to sublime - for now I will be sticking to Sublime however there are rumors that development on the editor has stopped so I could be jumping ship soon!

The last part of the introduction was a quick overview of the upcoming coursework and some extra reading around the HTML5 specification.
