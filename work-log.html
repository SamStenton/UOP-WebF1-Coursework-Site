Separating Form and Content

This week we are introduced to Cascading Style Sheets (CSS). By attaching a style sheet to my HTML I am able to change the web page to how I'd like it to look. First I need to create the extra document, I called my app.css. Next I linked the two together by placing a link tag in the head of my HTML. 


<!DOCTYPE html>
<html>
<head>a
	<title>Web Foundations Site </title>
	<link rel="stylesheet" href="app.css">
</head>

Once these were linked I was able to style any element on my page


div.demo{ 
	color: red;
	text-size: 2em;
	font-weight: bold;
	background: #fff;

}

Which creates the following:

This is the demo text

Although I have used an external style sheet it is possible to insert CSS style straight into the HTML document.


<div class="demo" style="color: red; text-size: 2em; font-weight: bold;">
	This is the demo text
</div>

Or At the top of the document in style tags


<style>
    div.demo{ 
	    color: red;
	    text-size: 2em;
	    font-weight: bold;
	    background: #fff;	
    }
</style>

There are three main types of CSS selectors: IDs, Classes and Block elements. IDs should only be used once - the official HTML validate will throw an error if it detects more than one ID with the same name. Classes can be used in multiple places on a page. For example if you wanted to keep the same width throughout multiple elements on the page you'd give them all classes. Finally block elements are just HTML elements on the page like <body> or <nav>

Using Media

Media can take many different forms on the web; from images to videos to audio. HTML5 makes consuming media via the web a much easier process with built in tags that allow almost seamless output of formats.

To display an image on a page we use the <img> tag which includes a src field to the location of the image on the local storage and an alt tag which is used to describe the image. 


<img src="images/background.jpg" alt="The background Image">

The above outputs the following

The <img> tag allows some extra attributes within itself. The main ones being width and height. By setting both these attributes you can set the image to an exact size - There can be issues however. If the aspect ratio is changed the image could appear stretched or "smushed"

Before the HTML5 specification is it was very difficult to include video on a web page. This was because not all browsers supported the same implementation. Now there is a universal selector which has brought everything together

The <video> allows the use of multiple formats to support as many browsers and devices as possible. These can be included with the <source> element within the video. 


<video controls>
  <source src="video/video.mp4" type="video/mp4">
  <source src="video/video.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Graphics on The Web

With the advent of faster and faster internet the use of Media and Graphics is getting more popular. Along with images and video a large part is now the use of Scalable Vector Graphics which allows the use of Vector images in online media. Vector graphics are nothing new - they've been used in print for many years. The special thing about them is they keep their quality when scaled and have very small file sizes.

  • Small file sizes that compress well
  • Scales to any size without losing clarity
  • Looks great on large pixel displays
  • Can be dynamically created
